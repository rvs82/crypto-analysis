Техническое задание на создание сайта для криптовалютного трейдинга:
1. Общее описание: Сайт должен предоставлять рекомендации по торговле криптовалютами (LDO/USDT, AVAX/USDT, XLM/USDT, HBAR/USDT, BAT/USDT, AAVE/USDT) с использованием максимального количества технических индикаторов и искусственного интеллекта для анализа данных и принятия решений.
2. Функционал: Отображение текущих рыночных настроений: проценты "Лонги" и "Шорты" (целые числа, не более 100%). Таблица с данными по каждой паре: название пары, рекомендация (направление, вход, стоп-лосс, тейк-профит, уверенность, соотношение риск/прибыль), текущая сделка, количество закрытых сделок, стопов, профитов, общая прибыль и убыток в USDT. Открытие сделок только при уверенности не ниже 95% с высокой стабильностью тренда. Звуковое уведомление при открытии новой сделки (включение/выключение через чекбокс). Под таблицей: список всех используемых индикаторов и фильтров, принципы принятия решений об открытии сделок, параметры активных сделок (неизменные до закрытия).
3. Дизайн: Шрифт в таблице на 1 пиксель меньше стандартного (15px). Слово "Лонг" выделять зелёным цветом, "Шорт" — красным. Вместо "Стоп" использовать "С", вместо "Профит" — "П".
4. Технические требования: Использовать все возможные технические индикаторов для анализа (25 и более), включая трендовые, импульсные, волатильные, объёмные и уровневые. Применять искусственный интеллект (например, линейную регрессию) для прогнозирования цены и повышения точности. Учитывать ложные сигналы: фильтровать перекупленность/перепроданность, слабые тренды, высокую волатильность. Обновление данных каждые 5 секунд через WebSocket с Binance. Стабильность: сделки открываются только при подтверждённом сильном тренде с минимальными колебаниями.
5. Дополнительно: Сайт должен быть надёжным, без багов, с точными прогнозами, минимизирующими убытки. Все данные (индикаторы, принципы, сделки) должны быть доступны пользователю на главной странице.
